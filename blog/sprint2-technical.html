What are best practices associated with using classes vs ids? --- We need ways to describe content in a HTML document. The basic elements like h1, p, and ul will often do the job, but our basic set of tags doesn't cover every possible type of page element or layout choice. For this we need ID's and Classes. So what is the difference between class and id? ## ID's are unique Each element can have only one ID Each page can have only one element with that ID, For example

Any styling information that needs to be applied to multiple objects on a page should be done with a class. Take for example a page with multiple "outline", which gives awesome outline effect to the text. in my css file, I defined a class named outline, /** define the outline effect to text */ .outline { color: black; text-shadow: 0px 0px 6px gold; font-size: 60px; } in my html file, I can apply outline class as many times as you need to
