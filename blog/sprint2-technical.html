We need ways to describe content in a HTML document. The basic elements like <h1>, <p>, and <ul> will often do the job, but our basic set of tags doesn't cover every possible type of page element or layout choice. For this we need ID's and Classes.
So what is the difference between class and id?

ID must be unique and it looks like #elementID

Each element can have only one ID;
Each page can have only one element with that ID, in general, use id whenever you want to refer to a specific element, for example <ul id="noBullets">, this will give us the chance to target this unordered list specifically, so that we may manipulate it uniquely to other unordered lists on our page.

in a css file - define an id named nobullets
/**remove the bullet dots for that perticular unordered list*/

#noBullets{
list-style-type: none;
margin: 0;
padding: 0;
}

in a html file - apply this id to the particular ul
<ul id="noBullets">
<li>list1<li>
<li>list2<li>
</ul>

Or we might have a section on our page that has no relevant tag to signify it, for example a footer, where we might do something like this: <div id=â€œfooter">.

in a css file - define an ID named footer
/** define footer style */

#footer{
padding: 10px;
text-align: centre;
background: transparent;
margin-top: 10px;
color: black;
}

in a html file - apply footer ID to Footer only

<div id="footer"> footer contents</div>

Classes are NOT unique, class can be applied to many things, class elements looks like .className

You can use the same class on multiple elements, in general you use class when you have a number of things that are all alike.

For example a page with multiple "outline", which gives awesome outline effect to the text.

in a css file, define a class named outline,
/** define the outline effect to text */

.outline {
color: black;
text-shadow: 0px 0px 6px gold;
font-size: 60px;
}

in a html file, apply outline class as many times as needed

<div class="outline"> outline text 1 </div>
<div class="outline"> outline text 2 </div>
<div class="outline"> outline text 3 </div>

outline class have been repeatly applied in this html file.

You can use multiple classes on the same element.
for example

/** define a green-background class */

.green-background{
background-color: green;
}

then I can apply class outline that I defined previously and class green-background on the same element;

<div class="outline green-background"> contects here
</div>
